FreshPorts Development Site - mailing lists

develop

The develop list is where dicussion about features and problem solving occurs. If you want to help out, this is the list to join.

To join the FreshPorts development mailing list, please send a message to majordomo@FreshPorts.org with "subscribe develop" in the body of the message (without the quotes...).

cvs-all

The cvs-all list receives the cvs commit messages for the FreshPorts repository.

To join the FreshPorts development mailing list, please send a message to majordomo@FreshPorts.org with "subscribe cvs-all" in the body of the message (without the quotes...).
